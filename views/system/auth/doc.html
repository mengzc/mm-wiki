认证接口配置文档
内置LDAP 
认证域名称: LDAP域名称
认证URL: ldap://host:port
扩展数据: LDAP域Base，如dc=xxx,dc=com
HTTP认证 
认证域名称: HTTP虚拟域名称
认证URL: http://host:port/path
扩展数据: 按HTTP认证接口要求
HTTP认证接口开发文档
接口URL 统一登录认证接口地址（例如：http://intra.mmWiki.com/login/auth）
请求方式 POST
请求参数 
username: 用户名（string）
password: 密码（string）
ext_data: 登录认证扩展数据（string）
成功返回 必须为 json 字符串 
{
    "message": "", // 必须返回，成功时为空
    "data": {
        "given_name": "王哈哈", // 姓名，string，不能为空
        "mobile": "111111111111", // 手机号，string，可以为空
        "phone": "010-9929921", // 电话，string，可以为空
        "email": "root@mmWiki.com", // 邮箱，string，不能为空
        "department": "广告事业部.技术部.系统开发组", // 部门，string，可以为空
        "position": "高级JAVA开发工程师", // 职位，string，可以为空
        "location": "B座32层E区1002", // 工位位置，string，可以为空
        "im": "QQ：12211", // 即时聊天工具，string，可以为空
    } // 必须返回, 成功时非空
}
失败返回 必须为 json 字符串 
{
    "message": "密码错误", // 失败原因，失败时不能为空
    "data": {} // 失败时为空
}
